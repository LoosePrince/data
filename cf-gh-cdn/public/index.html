GitHub 文件中转

通过 Cloudflare Pages Function（Workers）携带服务端配置的 GITHUB_TOKEN 调用 GitHub Contents API，将文件写入指定仓库路径，并返回 jsDelivr 与 jsDMirror 的 raw 链接。默认目标仓库为 LoosePrince/SocialFlow，可自行修改。 



      

        

          部署：将 GITHUB_TOKEN 配为 Pages 生产环境 Secret；本地把 .dev.vars.example 复制为
          .dev.vars 并填写同名字段。然后 npm install、npm run dev 预览，
          npm run deploy 发布（需 npx wrangler login）。
        

        

          安全提示：Token 仅在 Worker 端使用，浏览器不会收到 PAT；但任意能打开该页面的人都能触发上传写库，请为站点开启
          Cloudflare Access、IP 限制或仅私有使用，并为 PAT 设置最小仓库权限。
